有环单链表

发表于 2019-09-02 更新于 2020-03-08 分类于 LeetCode ， LinkedList

kernel：快慢指针

判断是否有环

  • 快指针一次跳两步，满指针一次跳一步，有环就意味着总会相遇，即当slow=fast时意味有环

找出环的入点

  • 当两指针相遇后令fast=head，然后fast和slow都每一次走一步，直到再次相遇，相遇点即为环的入点

求环长

  • 环长即为从head到入点的距离

求单链表长度

  • 由上述已得到非环长度A，环长B，则单链表长度 = A + B

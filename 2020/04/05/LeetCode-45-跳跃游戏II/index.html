45. 跳跃游戏 II

Difficulty: 困难

给定一个非负整数数组，你最初位于数组的第一个位置。

数组中的每个元素代表你在该位置可以跳跃的最大长度。

你的目标是使用最少的跳跃次数到达数组的最后一个位置。

示例:

说明:

假设你总是可以到达数组的最后一个位置。

Solution

思路1：

记录当前可跳范围内跳的更远的位置中的最远的位置，每当到达可跳范围内最远的位置时，就增加一步，并把新一步的可跳范围更新为刚刚过去的范围中比较出来的跳的最远的位置。
Language: Java
时间复杂度：O(N)
空间复杂度：O(1)

思路2

从后向前，查找能到达当前位置的 最左边的位置，步数+1，再继续查找能到达这个位置的最左侧位置，直到能到达0这个位置。
Language：Java
时间复杂度：O(N^2)
空间复杂度：O(1)

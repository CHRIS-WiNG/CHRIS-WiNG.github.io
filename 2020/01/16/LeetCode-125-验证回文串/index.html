125. 验证回文串

Difficulty: 简单

给定一个字符串，验证它是否是回文串，只考虑字母和数字字符，可以忽略字母的大小写。

说明：本题中，我们将空字符串定义为有效的回文串。

示例 2:

Solution

Language: Java

Thoughts

采用对撞指针，获取每个位置的字符，确定以下几个步骤：

  1. 如果左指针指向的字符不为数字或字母，则跳过该字符
  2. 如果右指针指向的字符不为数字或字母，则跳过该字符
  3. 如果1、2均没有执行，则将两个字符都转换为小写后判断是否相等，如果不相等则直接返回false
  4. 如果没有返回false，则表示匹配成功，两个指针各向中间移动一位

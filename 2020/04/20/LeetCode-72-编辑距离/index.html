72. 编辑距离

Difficulty: 困难

给你两个单词 word1 和 word2 ，请你计算出将 word1 转换成 word2 所使用的最少操作数 。

你可以对一个单词进行如下三种操作：

  1. 插入一个字符
  2. 删除一个字符
  3. 替换一个字符

Thought

  1. 状态空间：
    • 字符串问题一般是二维状态数组
    • dp[m + 1][n + 1]：word1[0...m]转换为word2[0...n]的最少步数
  2. 状态转移
    • 子问题：dp[i][j]有几种方式能从上一个状态转移过来：
      1. word1[i]修改为字符word2[j]，可转换为word2，之前要做的就是要将word[0...i-1]转换为word2[0...j-1]（dp[i - 1][j - 1]）。
      2. word1[i]后增加一个字符word2[j]，可转换为word2，之前要做的就是要将word1[0..i]转换为word2[0...j-1]（dp[i][j - 1]）。
      3. 删除word1中的word1[i]，可转换为word2，之前要做的就是要将word1[0...i-1]转换为word2[0...j]（dp[i - 1][j]）。
      4. word1[i]与word2[j]相等，不用修改任何字符，之前要做的就是将word1[0...i-1]转换为word2[0...j-1]（dp[i - 1][j - 1]）。
    • 转移方程：
      • dp[i][j] = min {dp[i - 1][j - 1] + 1, dp[i][j - 1] + 1, dp[i - 1][j] + 1, dp[i - 1][j - 1] && word1[i - 1] == word2[j - 1]}
  3. 初始值：
    • 空串和空串的最小编辑距离是0
    • 一个空串和一个长度为L的串的最小编辑距离是L（也就是让空串一直插入一个字符与L匹配）
    • 时间复杂度：O(m*n)，空间复杂度：O(m*n)
